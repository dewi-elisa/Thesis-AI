The protein alignment dataset is obtained from Pfam, selecting from the Pfam
multi-sequence alignments only the files with only two sequences.

https://pfam.xfam.org/about

Pfam version 35.0 was produced at the European Bioinformatics Institute using a
sequence database called Pfamseq, which is based on UniProt release 2021_03.
Pfam is freely available under the Creative Commons Zero ("CC0") licence,
https://creativecommons.org/publicdomain/zero/1.0/.
